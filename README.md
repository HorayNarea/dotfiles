# dotfiles

#### Installation
```sh
git clone git@github.com:HorayNarea/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```

#### Dependencies
* git
* zsh
* [exa](https://the.exa.website/) or ls from GNU Coreutils

#### recommended Tools
* sudo
* tmux
* mosh
* tree (only if exa is not installed)
* htop
* nano
