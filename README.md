# dotfiles

#### Installation
```sh
git clone git@github.com:HorayNarea/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```

#### Dependencies
* git
* zsh
* [eza](https://github.com/eza-community/eza) or ls from GNU Coreutils

#### recommended Tools
* sudo
* zellij
* mosh
* tree (only if exa is not installed)
* htop
* nano
